# Adobe Substance 3D Painter — Python API

## Documentación oficial

- **Sitio de referencia (Adobe Docs / GitHub Pages)**: [Substance 3D Painter Python API](https://adobedocs.github.io/painter-python-api/)
- **Adobe Help Center**: [Substance 3D Painter Python](https://helpx.adobe.com/substance-3d-painter-python.html) (guías y enlaces a API).
- **API index (Help)**: [API](https://helpx.adobe.com/substance-3d-painter-python/api.html)

## Capacidades habituales

- Plugins Python en la carpeta de documentos del usuario (`python/plugins`).
- Integración con exportación de texturas, proyectos, texture sets; en algunos flujos hay puente con **JavaScript** legacy vía módulo documentado.

## Tooling

- Revisar **versión del API** indicada en la documentación de tu versión instalada de Painter.
- Variables de entorno para rutas de plugins y `PYTHONPATH` (ver guías “loading external modules”).

## Notas para catálogo de herramientas

- Anotar **versión de Substance Painter** objetivo y si el plugin usa solo Python o también evaluación JS.
